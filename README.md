# Project 00: Racing Game

The goal of this is to have a two player game in which players mash buttons to try to get to the other end.  First player across the board is the winner.

## Technologies Used

- Javascript / jQuery
- Bootstrap
- CSS
- HTML

## Existing Features

- Two players
- Score board
- Countdown to start
- Reset button

## Planned Features

- Graphics
- Rocks and stuff to avoid
